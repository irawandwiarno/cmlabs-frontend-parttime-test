# Meal App

This project is a **part-time job technical test submission for CMLabs**.

A modern recipe browsing web application built with **Next.js 16**, **React 19**, **TypeScript**, **Tailwind CSS v4**, and **TheMealDB API**.

This project allows users to:
- browse a large list of ingredients
- explore meals based on selected ingredients
- view meal details, instructions, tutorials, and recipe compositions
- search with typo-tolerant matching
- enjoy dark/light mode support and smooth UI interactions

---

## 🚀 Tech Stack

- `Next.js 16`
- `React 19`
- `TypeScript`
- `Tailwind CSS v4`
- `Redux Toolkit`
- `Axios`
- `Framer Motion`
- `Lucide React`
- `TheMealDB API`

## 📦 How to Clone the Project

```bash
git clone https://github.com/irawandwiarno/cmlabs-frontend-parttime-test.git
cd cmlabs-frontend-parttime-test
```

---

## ▶️ How to Run the Project

### 1. Install dependencies

```bash
npm install
```

### 2. Create environment file

Create a `.env.local` file in the project root and add:

```env
NEXT_PUBLIC_MEALDB_BASE_URL=https://www.themealdb.com/api/json/v1/1
```

### 3. Run development server

```bash
npm run dev
```

### 4. Open in browser

Visit:

```bash
http://localhost:3000
```
